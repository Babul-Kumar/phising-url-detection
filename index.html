How PhishGuard Works

A 3-stage pipeline ensures accurate, real-world reliable detection

⚙️
Feature Engineering
19 hand-crafted features: URL length, entropy, subdomain count, suspicious keywords, IP detection, and more. Schemes are stripped before computing to match training data.
🌲
Random Forest Model
Trained on 549K+ real phishing and legitimate URLs. GridSearchCV-tuned hyperparameters, threshold optimised on the Precision-Recall curve for minimum false positives.
🏛️
Hybrid Domain Awareness
Known-trusted domains (Google, GitHub, etc.) get a whitelist pass, preventing false positives on well-known sites even if URL features look suspicious.
🔬
Explainable AI
Every prediction shows the top 5 features driving the decision — so you understand why a URL is flagged, not just that it is.
